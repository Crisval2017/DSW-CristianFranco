Sitio web para el aprendizaje de Flex Box y CSS Grid

Este sitio tiene como propósito suplir la necesidad de la presencialidad
en las instituciones educativas, de acuerdo a esto, su objeto es dar a
conocer temas como Flex Box y CSS Grid, explicando las principales
características de cada uno, para que pueden servir y en qué casos
es más idóneo usarlos.

Flex Box

Flexbox es un modelo flexible que permite organizar los elementos que contiene una página
según cambien las dimensiones y orientación de la misma. Es por esto, que facilita hacer
un diseño adaptativo para que las páginas web se visualicen correctamente en cualquier
dispositivo. Se puede decir que este modelo es unidimensional, es decir, para diseños
de una fila o una columna.



CSS Grid

CSS Grid, más conocido como modelo de rejilla al igual que flexbox se usa para controlar
el posicionamiento de los elementos en las páginas web, en este caso la pantalla se
divide en una especie de rejilla virtual y es posible elegir cuantas celdas de dicha
malla o rejilla puede ocupar cada elemento. Se puede decir que este modelo es
bidimensional, ya que sus diseños en varias filas y columnas al mismo tiempo.
